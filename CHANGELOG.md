# BazFlightZoom Changelog

## 006 - Unified Profiles
- Profiles now managed centrally in BazCore settings
- Removed per-addon Profiles subcategory

## 003 - Audit Fixes
- Removed manual profile proxy (now auto-wired by BazCore)
- Category changed to "Baz Suite"

## 002 - BazCore Migration + New Features
- Migrated to BazCore framework (profiles, settings panel, minimap button)
- Configurable camera distance (5-50, default max)
- Smooth zoom speed option (instant or gradual transition)
- Configurable zoom delay after mounting
- Minimap zoom level slider (instead of always fully out)
- Optional ground mount zoom with separate distance setting
- Profile support for per-character settings
- BazCore is now a required dependency

## 001
- Initial release
- Auto zoom out camera and minimap when flying
- Restore previous zoom levels on dismount
- Flying mount detection only (ignores ground mounts)
- Settings panel with per-feature toggles
